## 🐛 Debugging Tips

### Console Logging
```csharp
Console.WriteLine($"Entity {entity.Id}: HP {hp.Current}/{hp.Max}");
```

### Visual Debugging
```csharp
// Desenhar bounding boxes de colisão
#if DEBUG
    DrawDebugRect(entity.Position, entity.Size, Color.Red);
#endif
```

### Performance Profiling
```csharp
var sw = System.Diagnostics.Stopwatch.StartNew();
// ... código a medir ...
sw.Stop();
Console.WriteLine($"Tempo: {sw.ElapsedMilliseconds}ms");
```
